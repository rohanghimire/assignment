Learning experience of Web Development

There were few ups and downs while learning Web Development and we did remove all the obstacles while Learning. As per my view, we had a great learning environment at NAMI and as for web development, it was the best learning experience than any other subject we have read yet. We went from bottom(Github, Github commands, pushing code in Github) to the peak of HTML and CSS. There was not much issue while learning HTML, CSS because we used to have more Tutorial classes. All the things taught in the lecture class were completed in the tutorial class within the week, So we didn’t had many issues while Learning. We did face some issues while pushing code to GitHub, Creating HTML pages, Writing CSS codes but all of them were solved in the Tutorial class by the modular leader. There were many expectations while studying web development, Though many of the expectations were reached I did had few expectations for the term(i). Firstly I thought we will be learning all the things of responsive view to make the website look beautiful in mobile/desktop view. We did learn about the responsive view but we were just given few ideas and taught to fire media query. From whats given in the assignment I thought we will also be learning few javascript codes in term(i) so everyone could have done much better in making the website responsive, beautiful. So, the certain expectation were not met according to me but I have tried my best to make the website unornamented/simple Though certain things are missing in the website as per my view and will be doing my best in term(ii).Besides certain things, I enjoyed learning about HTML and CSS a lot.
